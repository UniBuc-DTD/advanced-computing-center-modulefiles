# Advanced Computing Center modulefiles

## Description

This repository contains [module definition files](https://modules.readthedocs.io/en/latest/modulefile.html) for the [Environment Modules](https://envmodules.io/) setup on the High-Performance Computing cluster of the [University of Bucharest](https://unibuc.ro/?lang=en)'s [Advanced Computing Center](https://edis.unibuc.ro/data-center-si-laboratoare-de-acces/).
